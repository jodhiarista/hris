Story behind the features

Odoo behavior : Odoo proposes one general pipeline and split opportunities via the sales team notion.

CLuedoo behavior : Cluedoo proposes to manage business type thanks to a multi-channel CRM to split the notion of Sales team and the notion of Business type, and create dedicated pipeline for each business type.

Module + : Using the same CRM, the same database but in different way: you get the advantages of working together without the drawbacks to find a 100% common working process.

Business Case : Classic Multi-CRM is to split B2B and B2C activities because the way to communicate, get info are different.

CLuedoo recommendation : To be installed if you have different business with different sales approach.

Tips: Combined with Studio, this notion of business type give the possibility to create extra fields on the CRM which will be visible only according to the business type, and so to have customized CRM for each channel. Moreover, Cluedoo proposes also this notion on the Sales side in order to have multi-channel from CRM to Sales.

Our Feature as Solution

Create dedicated pipeline for each business type.

How to Use this Module?

1. Go to Settings > Users & Companies > Business Type. Create a new business type, give the name then set prefix and suffix.
2. We can also create menu for this business type.
3. Create name, select menu action and parent menu (where the menu will be shown up).
4. Now go to CRM > Sales, we can find the menu of the business type that we just created.
5. After creating business type, on user, set Allowed Business Type.
6. When we create an opportunity in Multi-CRM menu, the user can only create with allowed business type, and lead code automatically created based on sequence of the business type.
